# CENNA School Management System

A complete School Management System built using

- Next.js 15
- React
- Redux Toolkit
- Tailwind CSS
- Node.js
- Express
- MongoDB
- Cloudinary
- JWT Authentication

---

## Features

### Admin

- Dashboard
- Students
- Teachers
- Parents
- Classes
- Subjects
- Attendance
- Results
- Fees
- Timetable
- Datesheet
- Gallery
- News
- Notifications
- Settings

### Student

- Dashboard
- Profile
- Attendance
- Results
- Fees
- Timetable
- Datesheet
- Subjects
- Notifications

### Teacher

- Dashboard
- Attendance
- Results
- Timetable
- Students

### Parent

- Dashboard
- Child Attendance
- Child Results
- Fee Status

## Installation

### Backend

```bash
cd cenna-backend
npm install
npm run dev
```

### Frontend

```bash
cd cenna-frontend
npm install
npm run dev
```
